# Skills

Personal agent skills I use.

## Install

Use the skills installer from the target project or agent environment:

```bash
npx skills@latest add andreichenchik/skills
```

Then select the skills you want to install and the agents you want to install them for.

## Use

After installation, invoke a skill by asking the agent to use it, or by using the slash command created by your agent/skills installer when supported.

## Layout

Skills live under:

```text
skills/<skill-name>/SKILL.md
```

Each `SKILL.md` contains the skill metadata and the workflow instructions the agent should follow.
